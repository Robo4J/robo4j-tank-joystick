## Robo4J-tank-joystick-example
Robo4J Tank Joystick Demo example

Example uses JavaFX application with Robo4J system initiated in declarative way.
JavaFX event are send to Robo4J system. Robo4J system communicates with another 
Robo4J End-Points

http client has be configured over XML file
http client uses POST

## Building from Source
The Robo4j framework uses [Gradle][] to build.
It's required to create fatJar file and run.

## Requirements
* [Java JDK 8][]
* [Robo4j.io][] :: version: alpha-0.3
